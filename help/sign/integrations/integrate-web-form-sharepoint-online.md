# 整合網頁表單 [!DNL SharePoint Online]

瞭解如何將從網頁窗體收集的數據對應到 [!DNL SharePoint] 清單中，以便共用跨團隊收集的數據。

>[!NOTE]
>
>Acrobat網頁表單和 [!DNL SharePoint] 清單之間的欄位對應僅適用於在線安裝，而不適用於 On-Premises 安裝。
