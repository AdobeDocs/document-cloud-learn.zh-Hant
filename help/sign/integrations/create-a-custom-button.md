# 在 Acrobat Sign 中建立自訂按鈕 [!DNL Salesforce]

了解如何在 Salesforce.com 的標準物件中建立自訂按鈕。此範例中，自訂按鈕會新增至 **「連絡人** 」中的 [!DNL Salesforce] 。 按下按鈕後，就會啟動傳送程式，並將「連絡人 **」中的資訊** 直接填入合約中。
