# 使用 [!DNL Power Automate]

瞭解貴組織的非技術成員如何使用 Acrobat Sign 中免費提供的預建 [!DNL Microsoft Power Automate] 連接器，建立連接其他系統的簡易工作流程。 這些自動化的檔工作流程有助於提高一般流程的速度和效率、縮短花費在重複任務上的時間，並將錯誤和重做最小化。
