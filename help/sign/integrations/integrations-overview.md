# 整合總覽

您可以在貴組織已使用的其他應用程式中使用Acrobat Sign，例如Microsoft、Salesforce、Workday和Marketo等。 透過這些整合指南和教學課程，瞭解如何簡化您的電子簽章工作流程。

>[!NOTE]
> 如果您無法存取其中任何功能，請洽詢組織的管理員，以確保啟用整合。

## 新增功能

>[!BEGINTABS]

>[!TAB 設定大型檔案與推播對應]

瞭解如何在Salesforce中設定[大型檔案和推播合約服務](salesforce-large-files.md)。

>[!TAB 設定自動更新]

瞭解如何輕鬆將您的帳戶連結至Salesforce，以取得[自動更新](salesforce-automatic-updates.md)。

>[!TAB Microsoft Teams的檔案索引標籤整合]

瞭解如何直接從[!DNL Microsoft Teams]中的[檔案索引標籤](acrobat-sign-teams-documents-tab.md)傳送檔案以索取簽名。

>[!ENDTABS]
