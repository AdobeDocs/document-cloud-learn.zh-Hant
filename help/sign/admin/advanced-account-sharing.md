# 進階帳戶共用

瞭解如何設定進階帳戶共用 — 允許管理員和使用者委派其傳送、修改和檢視許可權。 帳戶共用最適合使用者休假或請假的情況。 或者，您可以建立虛擬使用者帳戶（例如HR管理使用者帳戶），然後將許可權授予HR群組內的多個使用者，這些使用者都可以代表虛擬使用者執行動作。

>[!NOTE]
>
>進階帳戶共用僅適用於企業級計畫。 「技術支援」可以啟用「進階共用」，但一旦啟用，就會以無法回覆的方式變更共用物件的關係。 因此，建議您連絡您的Adobe帳戶團隊，並建立試用帳戶以測試功能，並確認其允許您完成所有想完成的動作。
