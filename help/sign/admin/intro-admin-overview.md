# 管理員概觀

瞭解如何將使用者新增至您的帳戶、設定群組、共用存取權，以及設定工作流程、外部封存，以及共用事件和快訊。 這些教學課程是專為已過安裝階段並準備開始管理 Acrobat Sign 的管理員所設計。 進階工作涵蓋的主題會從管理員設定擴展到定義全域設定、帳戶存取和共用，以及文字標籤。
