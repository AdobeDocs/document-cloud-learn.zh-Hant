# 管理員總覽

![Sign 管理員影像](../assets/Hero-Admin.png)

瞭解如何將使用者新增至您的帳戶、設定群組、共用存取權，以及設定工作流程、外部封存以及共用的事件和警告。 這些教學課程是專為已進入安裝階段且已準備好開始管理 Acrobat Sign 的管理員所設計。 「進階工作」涵蓋從管理員設定擴展並開始定義全域設定、帳戶存取和共用以及文字標記的主題。
