# 管理員概觀

瞭解如何將使用者新增至您的帳戶、設定群組、共用存取權，以及設定工作流程、外部封存和共用事件與警示。 這些教學課程是專為已度過安裝階段並準備好開始管理Acrobat Sign的管理員所設計。 進階工作涵蓋的主題已超出管理員設定，並進入定義全域設定、帳戶存取與共用，以及文字標籤。

## 新增功能

>[!BEGINTABS]

>[!TAB 設定並控管修改協定]

瞭解如何在Acrobat Sign中設定及控管[修改合約](configure-modify-agreement.md)。

>[!TAB 建置自訂工作流程]

瞭解如何建立並使用[自訂工作流程](building-a-custom-workflow.md)，以加速建立和傳送合約的程式。

>[!TAB 建立可重複使用的範本]

瞭解如何建立[可重複使用的檔案範本](../sign-advanced-users/create-a-template.md)。

>[!TAB 啟用自動使用者指派規則]

瞭解如何設定[自動指派規則](automatic-assignment-rules.md)，以授與使用者在Admin Console中存取Acrobat Sign的許可權。

>[!ENDTABS]
