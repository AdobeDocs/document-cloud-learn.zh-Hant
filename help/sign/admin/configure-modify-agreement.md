# 設定並控管修改協定

瞭解如何在Acrobat Sign中設定及控管&#x200B;**修改合約**。 在本教學課程中，管理員會設定帳戶和群組層級控制項、瞭解適用性規則，並瞭解在傳送後進行變更時，如何保留合約完整性。 瞭解如何追蹤和稽核寄件者動作，以支援合規性和透明度。

您將瞭解的內容：

* 哪些管理員設定可啟用或限制協定修改
* 當協定可以和無法修改時
* 修訂模式如何保護協定完整性
* 稽核報告會記錄哪些變更
