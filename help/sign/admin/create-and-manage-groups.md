# 建立和管理群組

瞭解如何建立群組、將使用者新增至群組，以及編輯群組設定。 群組可讓您分別擁有可存取不同資料庫檔和不同設定 （例如標誌和副本收件者） 的使用者子集。

>[!NOTE]
>
>群組僅適用於業務和企業層級計劃。
