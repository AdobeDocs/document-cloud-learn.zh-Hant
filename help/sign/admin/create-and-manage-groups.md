# 建立和管理群組

瞭解如何建立群組、將使用者新增至群組，以及編輯群組設定。 群組可讓您擁有可存取不同元件庫文件和其他設定 (例如，標誌和已傳送副本的使用者) 之使用者的子集合。

>[!NOTE]
>
>群組僅適用于業務和企業層級計畫。
