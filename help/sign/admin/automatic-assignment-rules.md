# 啟用自動用戶分配規則

瞭解如何設定自動分配規則，以在[Admin Console](https://adminconsole.adobe.com/tw)中授予用戶訪問Acrobat Sign的權限 — 簡化設定並減少手動管理任務。 用戶可以通過從Admin Console生成的產品訪問URL邀請其他用戶到Acrobat Sign。
