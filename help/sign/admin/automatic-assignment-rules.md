# 啟用自動使用者指派規則

瞭解如何設定自動指派規則，以授予使用者在[Admin Console](https://adminconsole.adobe.com/tw)中Acrobat Sign的存取權 — 簡化布建並減少手動管理工作。 使用者可透過Acrobat Sign產生的產品存取URL，邀請其他使用者前往Admin Console。
