# 數位ID概述

數位身分（數位ID）類似於電子護照的形式，可讓您安全地證明自己的身份。 此外，在Acrobat Sign中執行電子簽章時，使用數位ID可讓您更高層級地確保已針對該特定檔案授權您的電子簽章。 下列教學課程會示範如何透過Acrobat Sign使用全球的數位ID。

>[!NOTE]
>
>若要將數位ID與雲端簽名搭配使用，請洽詢貴組織的管理員，確認已在Acrobat Sign中啟用提供者的解決方案。
