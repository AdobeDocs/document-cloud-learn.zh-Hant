# 移動友好視圖

完成旅途中的任務不再是可選任務 — 從專業人員簽訂合同到家長填寫學校表格，對簡單移動解決方案的需求比以往任何時候都大。 瞭解如何使用移動友好視圖來消除打印和縮放，並減少在移動設備上填寫表單時的錯誤。

有了移動視圖，Acrobat Sign讓任何人在移動設備上填寫和簽名表格變得比以往任何時候都更容易，無論表格在哪裡。 通過自動建立並在移動設備上提供簡化的UI,Mobile View可指導用戶完成必要的表單域，從而更輕鬆地通過以下方式高效準確地完成文檔：

* 適合移動的欄位：不再縮放和縮放，表單完全適合您的螢幕
* 提高的準確性：減少錯誤並確保捕獲所有必要資訊
* 連續性：用戶可以隨時開始並返回移動友好形式，而不會丟失進度
