# 建立移動友好視圖

瞭解如何使用「移動焦點」模式建立文檔的移動友好視圖。 發件人現在可以無縫地生成一種移動友好型格式，而無需任何開發人員支援。

使用Mobile Focus，發送者可從以下方面獲益：

* 完成率提高：表格完成率提高，結果改善
* 減少時間和成本：避免將文檔轉換為移動友好格式所需的工作
* 無需技術專業知識：以最少的工作量生成動態、移動友好的表單
