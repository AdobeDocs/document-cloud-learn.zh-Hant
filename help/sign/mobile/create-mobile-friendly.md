# 建立適合行動裝置的檢視

瞭解如何在沒有任何開發人員支援的情況下，順暢地產生適合行動裝置的檔案。

使用「行動檢視」，傳送者將受益於以下專案：

* 提高完成率：提高表單完成率可改善結果
* 減少時間和成本：避免將檔案轉換為適合行動裝置的表單時所需的工作
* 無需專業技術：輕鬆產生動態、適合行動使用的表單
