# 建立移動友好視圖

瞭解如何在不提供任何開發人員支援的情況下無縫生成移動友好文檔。

使用Mobile Focus，發送者可從以下方面獲益：

* 完成率提高：表格完成率提高，結果改善
* 減少時間和成本：避免將文檔轉換為移動友好格式所需的工作
* 無需技術專業知識：以最少的工作量生成動態、移動友好的表單
