# 何時使用建立副本

已傳送合約 — 但合約已過期、已取消，或您必須再次傳送？ 建立副本可讓您複製任何現有的協定、調整需要變更的內容，並在數秒後重新傳送。 不從頭重建。 建立復本可在下列情況下使用：

* 您是合約的原始寄件者
* 合約是從「現代要求簽名體驗」傳送
* 合約不包含見證收件者角色
* 由於檔案保留或GDPR刪除，合約尚未刪除
* 此合約不需要存取資料庫範本，寄件者無法再存取

>[!NOTE]
>
>建立復本可供企業級計畫使用。
