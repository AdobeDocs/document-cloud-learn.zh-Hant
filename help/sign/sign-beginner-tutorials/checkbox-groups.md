# 使用核取方塊群組引導所需的選取專案

Learn how to use checkbox groups to control how many options a signer can select on a form. Checkbox groups help ensure signers complete documents correctly by enforcing simple rules — such as requiring a minimum or maximum number of selections — before an agreement can be submitted.

Learn how checkbox groups:

* Prevent incomplete or invalid submissions
* Guide signers with clear, inline feedback
* Improve the accuracy of information collected in forms
