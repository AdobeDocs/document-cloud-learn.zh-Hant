# 取代簽署者

瞭解如何替換簽名者 — 例如，在傳送檔案以供簽名時使用了錯誤的電子郵件。

>[!NOTE]
>
>「取代收件者」的預設設定是以新收件者完全取代目前的收件者。 商業與企業級計畫具有其他可用選項。
