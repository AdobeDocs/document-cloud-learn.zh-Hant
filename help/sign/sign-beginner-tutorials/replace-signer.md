# 取代簽署者

瞭解如何取代簽署者，例如傳送檔以供簽署時使用錯誤的電子郵件。

>[!NOTE]
>
>「取代收件者」的預設設定是以新收件者完全取代目前的收件者。 商業版和企業級計畫有其他可用選項。
