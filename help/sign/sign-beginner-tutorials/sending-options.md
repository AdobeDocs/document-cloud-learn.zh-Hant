# 設定傳送選項

瞭解如何在傳送檔案以索取簽名時自訂協定詳細資料、附加檔案及設定其他設定，例如提醒、有效期和收件者語言。

>[!NOTE]
>
>您的管理員可能已自訂傳送選項，因此您看到的選項可能與本教學課程中顯示的選項不同。
