# 配置發送選項

瞭解如何在發送文檔進行簽名時自定義協定詳細資訊、附加檔案以及配置其他設定，如提醒、過期和收件人語言。

>[!NOTE]
>
>您的管理員可能已自定義了發送選項，因此您看到的選項可能與本教程中顯示的選項不同。
