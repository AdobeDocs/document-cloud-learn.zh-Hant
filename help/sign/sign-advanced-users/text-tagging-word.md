# 使用文字標記登入 [!DNL Microsoft Word]

瞭解如何透過新增「Acrobat簽署文字標籤 [!DNL Microsoft Word] 」來建立可重複使用的檔範本。 傳送檔進行簽名時，文字標記功能會自動在檔上建立表單欄位，無需手動放置欄位。 此程式可大幅減少修訂和變更 [!DNL Microsoft Word] 傳送以供簽署的檔所需的時間。
