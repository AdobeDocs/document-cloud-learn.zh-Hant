# 在[!DNL Microsoft Word]中使用文字標籤

瞭解如何在[!DNL Microsoft Word]中新增Acrobat Sign文字標籤，以建立可重複使用的檔案範本。 文字標籤在檔案送出供簽名時，會自動在檔案上建立表單欄位，不需要手動放置欄位。 此程式可大幅減少您傳送出去供簽名的[!DNL Microsoft Word]份檔案進行修訂與變更所需的時間。
