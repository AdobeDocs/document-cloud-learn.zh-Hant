# 設定簽署順序

瞭解如何為多個簽署者設定簽署順序。 按順序和/或並行方式傳送檔案，或傳送給特定的個人群組。

>混合路由僅適用於企業級計畫。 您的Acrobat Sign管理員可能需要在「帳戶設定」的「傳送設定」區段中啟用這些功能。
