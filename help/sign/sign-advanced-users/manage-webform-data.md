# 管理網頁表單資料

瞭解如何從 [ 網頁窗 ](webform.md) 體追蹤、管理和匯出資料。 本教學課程會顯示其他參與者的網頁表單工作流程，以及如何將多個資料項目目合併到單一 .csv 檔案中，以供檢視和分析。

>[!NOTE]
>
>您的 Acrobat Sign 管理員必須啟用「 **全域設定** 」「網頁表單 **」區段** 中 **的其他參與者** ，才能使用其他參與者工作流程。
