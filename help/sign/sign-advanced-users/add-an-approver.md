# 使用核准者角色

瞭解如何透過核准程式傳送檔。 在這個範例中，瞭解如何透過人力資源、潛在員工和雇用經理傳送雇用合約。 如果有人需要在傳送文件給簽署者之前先獲得他人確認，就適合使用核准者角色。
