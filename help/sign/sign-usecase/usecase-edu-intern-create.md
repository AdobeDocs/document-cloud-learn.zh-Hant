# 建立教育版網頁窗體實習申請

瞭解如何建立實習網頁窗體，讓學生能夠存取可從任何裝置和任何地點提交的必填表格。 此範例中，每個學生的工作流程都是依其學習課程而異，因為每位學生都需要新增自己的教職員總監和系主任來進行核准。
