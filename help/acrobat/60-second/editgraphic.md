# 在PDF中編輯圖形

在這段60秒鐘的教學影片中，瞭解如何使用Illustrator在PDF中對圖形進行進階編輯！ 選取「編輯PDF」後，在圖形上按一下滑鼠右鍵，然後選擇在Illustrator中編輯圖形。 完成後，只要選擇「在Illustrator中儲存」，Acrobat就會更新PDF中的圖稿。
