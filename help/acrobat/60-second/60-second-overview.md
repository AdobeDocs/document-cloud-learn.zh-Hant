# 60秒Acrobat概觀

60秒Acrobat提供一些小型教學課程，協助您在一分鐘或以內學習Acrobat中的新訣竅。 這些以任務為基礎的秘訣可協助您瞭解使用PDF檔案的新技能，方法是解除鎖定部分Acrobat的隱藏寶藏。 您可以觀賞其中一個來快速獲得解答，或觀賞五個來提升您的檔案生產力，同時還有時間享受您的咖啡會。
