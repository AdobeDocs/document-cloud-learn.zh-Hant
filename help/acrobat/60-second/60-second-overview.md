# 60 秒Acrobat概覽

60 秒Acrobat為您提供一口大小的教程，説明您在一分鐘或更短的時間內學習Acrobat的新技巧。 這些基於任務的技巧可説明您通過解鎖一些Acrobat隱藏寶石來學習處理 PDF 檔的新技能。 您可以觀看一個以獲得快速答案，或者觀看五個以提高您的檔工作效率 - 並且仍然有時間享受您的咖啡休息時間。
