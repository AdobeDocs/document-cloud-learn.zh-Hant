# 在[!DNL Office]中為網頁建立PDF

瞭解如何在不離開[!DNL Microsoft Office]的情況下為網頁應用程式建立PDF檔案。 此附加元件需要適用於團隊的Acrobat訂閱或適用於企業的Acrobat訂閱。

>[!NOTE]
>
>適用於[!DNL Microsoft 365]的Adobe Acrobat包含可供所有使用者使用的免費功能（例如在PDF中檢視和搜尋）。 其他功能（例如建立和合併PDF）需要適用於團隊的付費Acrobat或適用於企業的Acrobat訂閱。
