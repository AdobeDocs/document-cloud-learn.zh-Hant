# Protect帶密碼的PDF檔案

瞭解如何[向PDF添加密碼](https://www.adobe.com/tw/acrobat/online/password-protect-pdf.html){target="_blank"}以保護其他人不開啟或[編輯PDF檔案](https://www.adobe.com/tw/acrobat/online/pdf-editor.html){target="_blank"}。 保護您的PDF檔案可防止未經授權的訪問、保護敏感資訊、維護文檔完整性，並幫助遵守某些行業法規。

>[!TIP]
>
>若要密碼保護多個PDF檔案，請嘗試使用[Action Wizard](../advanced-tasks/action.md)。
