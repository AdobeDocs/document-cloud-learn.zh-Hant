# 將檔案合併為單一 PDF

瞭解如何合併許多不同類型的檔案 （PDF、Word、Excel、PowerPoint、JPEG 等） 轉換為單一 PDF。 將檔案合併為單一 PDF 可讓您管理和控制傳送給其他人的內容，從而創造更好的體驗。
