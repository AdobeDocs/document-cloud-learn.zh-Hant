# 將多個檔案合併至單一PDF

若要提供最佳客戶體驗，內容的建立速度必須比以往任何時候都要快。 瞭解如何將許多不同型別的檔案(PDF、Word、Excel、PowerPoint、JPEG等)順暢地合併為單一PDF。 [將檔案合併為單一PDF](https://www.adobe.com/acrobat/online/merge-pdf.html){target="_blank"}可讓您管理並控制您傳送給其他人的內容，創造更佳的體驗。
