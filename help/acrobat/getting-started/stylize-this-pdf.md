# 將此PDF風格化

瞭解如何使用風格化此PDF透過使用Adobe Express [Adobe Express](https://express.adobe.com)支援的應用程式內工具套用字型建議、色彩主題、影像和設計元素，以改善PDF的視覺外觀。 本教學課程也說明如何調整版面、新增Adobe Stock或AI產生的影像，以及自訂文字樣式來建立精美、具有專業外觀的檔案。 此影片教學課程使用[新的Acrobat體驗](new-workspace.md)。

## 將此PDF風格化如何協助不同團隊

* **法律與法規遵循**

  法律團隊使用風格化此PDF，將一致的格式、印刷樣式和視覺效果套用至原則和指引檔案，使其看起來專業且易於閱讀，同時保持合規且無任何意義改變。

* **人力資源**

  HR團隊使用風格化此PDF，將原則檔案、上線材料和訓練資源轉換為視覺清晰、可親近的PDF，以改善理解和員工參與。

* **銷售**

  銷售團隊使用風格化這個PDF，套用品牌字型、色彩和影像，快速改善提案和簡報，協助提供使用者端可用的檔案，而不需返回設計工具。

* **行銷與通訊**

  行銷團隊使用風格化此PDF，透過調整版面配置、套用視覺主題和結合影像，確保PDF在視覺上跨行銷活動保持一致，從而使PDF符合品牌標準。
