# 將PDF轉換為不同的檔案格式

瞭解如何將PDF檔案轉換成不同的檔案格式，例如[Microsoft Word](https://www.adobe.com/tw/acrobat/online/pdf-to-word.html)、[PowerPoint](https://www.adobe.com/tw/acrobat/online/pdf-to-ppt.html)和[Excel](https://www.adobe.com/tw/acrobat/online/pdf-to-excel.html)。 也會說明如何將PDF轉換為[Adobe Express](https://express.adobe.com)以新增品牌設計元素。 當您將PDF轉換為其他檔案格式(如Microsoft Word)時，它會改善檔案工作流程效率，並在對長檔案進行大幅變更時節省大量時間。 此影片教學課程使用[新的Acrobat體驗](new-experience.md)。

>[!TIP]
>
>若要將多個PDF檔案匯出成其他格式，請嘗試使用[動作精靈](../advanced-tasks/action.md)。
