# 自動調整版面

瞭解新的編輯模式，其可在PDF中跨頁面重新安排內容並自動調整版面。 此影片教學課程使用[新的Acrobat體驗](new-experience.md)。

>[!NOTE]
>
>自動調整版面配置尚未支援所有型別的PDF內容或語言。
