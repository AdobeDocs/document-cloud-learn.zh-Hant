# 進階任務總覽

透過Acrobat中的這些進階工具，超越基本概念。 瞭解如何建立動作以自動化重複工作、移除敏感資訊、最佳化大型檔案、收集表單資料及產生可存取的PDF。 檢視這個專業策劃的[播放清單](https://experienceleague.adobe.com/en/playlists/acrobat-peform-advanced-tasks)，以便在Acrobat中執行進階工作。
