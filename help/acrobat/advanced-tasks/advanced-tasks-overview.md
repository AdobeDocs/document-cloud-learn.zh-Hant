# 進階工作概觀

使用這些進階工具Acrobat超越基本知識。 瞭解如何建立動作來重複瑣事、移除敏感性資訊、縮減和優化大型檔案、快速收集表格數據，以及建立具備通用輔助功能的 PDF 檔案。 查看此經專家精心策劃 [的播放清單](https://experienceleague.adobe.com/en/playlists/acrobat-peform-advanced-tasks) ，以在Acrobat中執行進階工作。
