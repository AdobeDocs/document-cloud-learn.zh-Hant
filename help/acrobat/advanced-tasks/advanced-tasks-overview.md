# 高級任務概述

在Acrobat，借助這些高級工具，超越基礎知識。 瞭解如何建立操作以重複常規任務、刪除敏感資訊、減少和優化大檔案、快速收集表單資料以及建立可普遍訪問的PDF檔案。 請查看此專業策劃的[播放清單](https://experienceleague.adobe.com/en/playlists/acrobat-peform-advanced-tasks)，以在Acrobat執行高級任務。

## 新增功能

>[!BEGINTABS]

>[!TAB 刪除敏感資訊]

瞭解如何永久刪除[敏感資訊](redact.md)。

>[!TAB 引導操作]

瞭解如何使用[引導操作](action.md)提高效率。

>[!TAB SEOOptimize PDF（搜索引擎優化）]

瞭解如何[優化PDF](optimizeseo.md)，以改進Web上的可發現性和搜索引擎排名。

>[!ENDTABS]
