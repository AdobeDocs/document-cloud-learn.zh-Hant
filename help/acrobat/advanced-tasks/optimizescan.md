# 最佳化掃描的檔案

瞭解如何在Acrobat中增強結果，以獲得更好的PDF檢視和搜尋體驗（無論您的檔案來自相機或掃描器）。

>[!TIP]
>
>如果您有Acrobat Pro （Premium或Studio），請在動作精靈工具中執行「最佳化掃描檔案」，以自動化此程式。
